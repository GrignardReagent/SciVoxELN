# SciVox ELN Pilot Sales Package

## Positioning

SciVox ELN is an on-prem/VPC electronic lab notebook for biotech R&D teams that
need faster bench capture without giving up audit discipline. Scientists can
plan experiments, dictate notes, photograph handwritten pages, review Observe
mode run summaries, sign records, and export audit-ready evidence.

Use the claim **audit-ready and validation-friendly**. Do not claim that the
software by itself makes a lab Part 11 or GxP compliant.

## Five-Minute Demo

1. Show project access: admin creates a project and adds a scientist.
2. Start from a plan and create a linked experiment.
3. Add a voice note, OCR note, or Observe run entry.
4. Review the entry, sign with password confirmation, and lock the experiment.
5. Export the experiment HTML/JSON package and audit CSV.
6. Show on-prem options: Docker, Caddy HTTPS, and Whisper transcription.

## Pilot Offer

- Duration: 30 to 60 days.
- Team size: 3 to 10 R&D users.
- Scope: non-GMP R&D notebook workflows.
- Deployment: customer-controlled server, workstation, or private VPC.
- Success criteria: faster record capture, searchable experiment history,
  successful export/audit review, and positive scientist usability feedback.

## Buyer Questions To Answer

- Where is the data stored?
- Can we run speech-to-text without cloud services?
- Can unauthorized users access other projects?
- Can records be exported if we stop using the product?
- What is covered by software controls versus customer SOPs?
- What validation evidence exists for the pilot build?

## Follow-Up Enterprise Features

- SSO/SAML/OIDC.
- Postgres deployment.
- Per-project inventory.
- PDF/ZIP signed export bundles.
- SMTP-backed password reset and email verification.
- LIMS/instrument connectors for priority customer systems.
